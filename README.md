# ShopEasy — E-Commerce Web Application

A full-stack e-commerce application with product catalog, cart, checkout, and admin order management.

## ✨ Features
- User authentication (Register/Login) with JWT
- Role-based access (Admin/User)
- Product catalog with categories
- Add to cart, quantity adjustment, checkout
- Order tracking with status updates (pending/shipped/delivered)
- Admin panel: add/delete products, manage all orders

## 🛠️ Tech Stack
| Layer | Technology |
|-------|-----------|
| Frontend | React.js |
| Backend | Node.js + Express.js |
| Database | MongoDB Atlas |
| Auth | JWT + bcrypt |

## 📦 Installation

### Backend
```bash
cd backend
npm install
node index.js
```

### Frontend
```bash
cd frontend
npm install
npm start
```
